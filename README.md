# Student Management System

A full-stack Student Management System built using Node.js, Express, MySQL and Bootstrap.

## Features

- Add Student
- View Student List
- Edit Student
- Delete Student
- Search Student

## Tech Stack

Frontend:
- HTML
- Bootstrap
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MySQL

## How to Run

1. Clone repository

git clone https://github.com/adarsh-trivedi-123/student-management-system.git

2. Install dependencies

npm install

3. Start server

node server.js

4. Open browser

http://localhost:5000
